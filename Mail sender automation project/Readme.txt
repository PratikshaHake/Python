This project performs the following tasks :
1.Accept the directory name from user
2.Scan the directory for duplicate files
3.Add all duplicate files in a separate log file
4.Delete the duplicate files.
5.Send the log file to the senders email provided by user.

Command to excecute this script :

> Application_Name Absolute_Path_of_Directory Senders_Email

Example :
>py MainFile.py Demo xyz@gmail.com
